webapp
==========================
> 使用 koa 创建一个基础的SPA应用

主要实现功能
* 用户注册登陆
* ins 爬虫

###目录
* [启动](#启动)
* [技术栈](#技术栈)


启动
--------------------------
```Bash
# 安装依赖包
$ npm install

# 开发环境启动
$ npm run dev

# 项目打包
$ npm run build

# 生产环境启动
$ npm run server

# 生产环境一键安装打包启动
$ npm run pord
```
